## ✅ Завершённые проекты

### 1. Student Manager (Консольное приложение)
Первое приложение для работы со списком студентов. Основы Python: списки, словари, функции.

### 2. Product Manager (Консольное приложение) — [Код](product_manager.py)
Управление товарами с продвинутой логикой.
- 📋 CRUD: добавление, удаление, редактирование.
- 🔍 Поиск: по названию, ценовому диапазону, самый дорогой/дешёвый.
- 💾 Сохранение в JSON с автозагрузкой.
- 📊 Аналитика: общая стоимость, фильтрация.

### 3. 📝 Notes App v3.0 — [Код](notes_app/notes.py)
**Production-ready** консольный менеджер заметок.
- ✅ **Полный CRUD:** создание, чтение, обновление, удаление.
- 🔍 **Умный поиск:** по заголовку, тексту, тегам (регистронезависимый).
- 📊 **Статистика:** количество заметок и тегов.
- 💾 **Автосохранение:** в JSON с обработкой ошибок.
- 🆔 **Уникальные ID:** сохранение между сессиями.
- 🎯 **User-friendly интерфейс:** редактирование с preview.

### 4. 💰 Finance Tracker v1.0 — [Код](finance_tracker/finance_tracker.py)
**ООП-проект** для учёта личных финансов.
- ➕ Добавление доходов и расходов с категориями.
- 📈 Подсчёт текущего баланса.
- 📋 Красивый вывод всех транзакций.
- 💾 Полное сохранение и загрузка данных в JSON (с поддержкой `datetime`).
- 🆔 Автоматическая генерация уникальных ID.

### 5. 🏃 Habit Tracker (Веб + SQLite) — [Сайт](https://learning-journal-tau.vercel.app/)
Полноценное веб-приложение на Flask.
- ✅ Добавление, выполнение, удаление привычек
- ✅ Умный streak (учитывает пропуски)
- ✅ Статистика на главной
- ✅ Хранение в SQLite
- ✅ Адаптивный интерфейс (CSS)
